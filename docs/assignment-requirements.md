# Assignment Requirements

## Goal
สร้างโปรแกรม todo-manager แบบ console application ด้วย TypeScript และ Bun

## Required Features
1. Add task
2. List tasks
3. Mark task as done
4. Remove task

## Input Constraints
- title ต้องไม่เป็นค่าว่าง
- id ต้องเป็นจำนวนเต็มบวก

## Output Expectations
- แสดงข้อความที่อ่านง่าย
- แสดงสถานะงานแบบ pending/done

## Learning Focus
- Primitive data types
- Type casting
- Arrays
- String processing
- Loops
- If-else
- Switch-case
- Enum or literal types
- Union type
- Type alias
- Functions
